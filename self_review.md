> ### [부록] AI 협업을 통한 주도적 트러블슈팅 및 문제 해결 과정
>
> 本 프로젝트를 수행하며 최신 개발 환경에서 발생한 복합적인 시스템 에러를 AI(Gemini)와의 유기적인 협업 및 디버깅을 통해 주도적으로 해결하였습니다.
>
> 1. **버전 호환성 예외 처리 및 Mock 단위 테스트 도입**
>    * **문제 상황:** 파이썬 최신 환경에서 MediaPipe 내부 모듈이 로드되지 않고 `AttributeError: module 'mediapipe' has no attribute 'solutions'` 에러가 발생하여 `pytest`가 실패함.
>    * **해결 과정:** 프로그램이 완전히 다운되는 것을 막기 위해 부모 클래스 내부에 `try-except` 예외 처리 구조를 선제적으로 반영함. 테스트 환경에서는 외부 환경에 의존하지 않도록 `unittest.mock` 객체를 주입하여 랜드마크 검출 로직을 시뮬레이션하는 전문적인 단위 테스트 코드를 작성, **단위 테스트 100% 통과(6 passed)**를 달성함.
>
> 2. **Git 형상 관리 오류 해결**
>    * **문제 상황:** 수정 사항 반영 중 변경 사항이 Staging Area에 담기지 않아 `git commit` 시 에러 발생.
>    * **해결 과정:** AI의 피드백을 통해 Git의 동작 메커니즘을 정확히 이해하고 `git add .` 명령어를 통해 정확히 장바구니에 담은 후 논리적인 단위로 단계별 커밋 히스토리(총 5개)를 구축함.
>
> 3. **OpenCV GUI 백엔드 충돌 해결**
>    * **문제 상황:** 하드웨어 권한 문제 우회를 위해 설치했던 헤드리스(Headless) 라이브러리로 인해 화면 출력 함수(`cv2.imshow`)가 튕기는 현상 발생.
>    * **해결 과정:** 에러 로그를 분석하여 헤드리스 버전의 한계를 인지하고, 일반 표준 버전인 `opencv-python`으로 재정렬하여 최종적으로 실제 물리 카메라 영상에 손 제스처 포인팅이 유기적으로 연동되는 메인 어플리케이션을 완성함.
