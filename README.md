# Hand Gesture Painter (손 인식 화면 표시 도구)

## 1. 프로젝트 개요
MediaPipe와 OpenCV를 활용하여 사람의 손을 인식하고, 손가락 제스처에 따라 화면에 실시간으로 그림을 그리거나 조준점을 이동할 수 있는 객체지향형 Python 패키지입니다.

## 2. 핵심 기능 및 제스처 원리
* **그리기 모드 (🔴 빨간색 점):** 검지 손가락 하나만 펼쳤을 때 (`☝️`) 실시간으로 화면에 선을 그립니다.
* **이동 및 대기 모드 (🟣 보라색 점):** 검지와 중지를 함께 펼쳤을 때 (`✌️`) 선을 그리지 않고 포인터만 이동합니다.

## 3. 기술 스택 및 구조
* **Language:** Python 3.14+
* **Libraries:** OpenCV, MediaPipe, pytest
* **Architecture:**
  * `HandDetector` (부모 클래스): 웹캠 영상 처리 및 손 랜드마크 추출 (캡슐화 및 비공개 메서드 `_preprocess_image` 활용)
  * `GesturePainter` (자식 클래스): 부모 클래스를 상속받아 `super()`를 활용하고 제스처 판별 로직 추가
  * `utils.py`: 두 점 사이의 유클리드 거리를 계산하는 단일 책임 원칙 기반 도우미 함수
